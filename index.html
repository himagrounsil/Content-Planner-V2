Modul Utama

Content Planner

Pusat kontrol konten. Anda bisa menambahkan task satuan (Tambah Task) atau banyak sekaligus (Tambah Banyak).

Prestasi Mahasiswa

Data otomatis dari Google Form. Gunakan filter Tingkat untuk memantau pencapaian.

Media Partner

Manajemen kerjasama. Cek status administrasi (Poster, Caption, Surat) dan hubungi CP via WhatsApp.

Fitur Menengah

  • Bulk Selection: Centang checkbox pada kartu task untuk memunculkan bar aksi masal (Ubah status atau hapus banyak).
  • Mode View: Gunakan List untuk daftar, Month untuk ringkasan bulanan, dan Calendar untuk visualisasi jadwal.
  • Auto Planner: Klik tombol "Auto Planner" di Media Partner untuk memindahkan data ke jadwal konten secara instan.

Aturan Database (Spreadsheet)

Sistem menggunakan Header Mapping. Jangan mengubah nama kolom kunci berikut:
Sheet Elemen Kunci (Headers) Aturan
Content Planner No, Task, Platform, Format, Due Date, Status Data utama dimulai dari Baris 17. Jangan hapus baris header.
Dropdown List UniqueSubDiv, UniqueFormat Gunakan sheet ini untuk menambah anggota tim atau format baru.
Prestasi / Media CMS_Status DILARANG MENGUBAH/MENGHAPUS kolom ini. Digunakan sistem untuk tanda PROCESSED.

Fitur Otomatisasi (Tanpa Campur Tangan)

Sistem backend akan bekerja otomatis pada jadwal berikut:

  • Prestasi Batch 1: Setiap tanggal 15 pukul 21:00. Mengangkut data baru menjadi task konten.
  • Prestasi Batch 2: Setiap tanggal 1 pukul 01:00 Pagi. Mengangkut sisa data bulan sebelumnya.
  • Media Partner H-7: Berjalan setiap hari. Membuat task otomatis jika publikasi tinggal 7 hari lagi.
